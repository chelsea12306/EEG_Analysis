# Electroencephalogram (EEG) Analysis
Brain signal processing, feature extraction, and classification for brain-computer interface (BCI) applications.

## Overview
This project provides a complete pipeline for EEG signal analysis, supporting both frequency domain analysis and motor imagery classification.

### Frequency Analysis
- Implements a full workflow for raw EEG preprocessing, noise reduction, and spectral feature extraction.
- Visualizes signal characteristics and frequency domain features for data interpretation.

### MI EEG
- Implements motor imagery (MI) classification tasks for brain-computer interface systems.
- Builds and optimizes machine learning models to recognize MI patterns from EEG signals.
- Validates model performance with standardized evaluation metrics.
